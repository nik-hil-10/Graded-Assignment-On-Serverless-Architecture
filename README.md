# Graded Assignment on Serverless Architecture

This repository contains the solutions, code, and documentation for 4 selected assignments on automating AWS infrastructure using AWS Lambda and Boto3. 

## Completed Assignments

1. **Assignment 1:** Automated Instance Management Using AWS Lambda and Boto3
2. **Assignment 4:** Automatic EBS Snapshot and Cleanup
3. **Assignment 5:** Auto-Tagging EC2 Instances on Launch
4. **Assignment 14:** Monitor EC2 Instance State Changes

Each folder contains the Python `lambda_function.py` code used as well as a `README.md` with detailed steps and screenshots of the execution in the AWS Console.
